Java Projects

Here are some of my highlighted Java projects that showcase my skills and creativity...

Egyptian War (April 2023)

A terminal-based card game inspired by the classic Egyptian War. Developed using Java Swing, this engaging game combines strategy and excitement in a unique gaming experience.

RPG Game (November 2023)

Refactored the code of a starter RPG game to develop our own unique version, featuring a turn-based battle system and a variety of music.

Quinnipiac Hackathon (Spring 2024)

Inspired by the theme of disability, I collaborated with a team to develop an engaging visual and auditory game using Java Swing, which earned us second place in the competition.

Trivia Game (April 2024)

Developed an engaging trivia game that leveraged both TCP and UDP protocols, enabling an unlimited number of clients to participate simultaneously.
